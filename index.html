InterPlanetary File System (IPFS) is a protocol and peer-to-peer network for storing and sharing data in a distributed file system. IPFS allows users to not only receive but host content. As opposed to a centrally located server, IPFS is built around a decentralized system of user-operators who hold a portion of the overall data, creating a resilient system of file storage and sharing.

Through an effort of Protocol Labs and Kiwix, with the support structure of the non-profit Wikimedia Foundation, we strive to bring the best of free human knowledge on the most powerful distributed Web network.
